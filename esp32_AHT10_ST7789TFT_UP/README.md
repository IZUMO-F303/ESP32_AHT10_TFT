# ESP32 環境時計 (AHT10 + ST7789 + NTP)

このプロジェクトは、ESP32 デバイスを使用して NTP サーバーから正確な時刻を取得し、同時に AHT10 センサーから温湿度を取得して表示するデジタル時計兼環境モニタです。

## 主な機能
- **高精度デジタル時計**: 
  - NTP サーバーから日本時間 (JST) を取得。
  - 画面上部に大型フォント (`Font7`) で「時:分」を表示。
  - バッテリーやリソース保護のため、表示更新は **1分に1回** (分が切り替わるタイミング) に最適化。
- **温湿度測定**: AHT10 センサーによる温度（摂氏）と相対湿度の測定。
- **絶対湿度計算**: テトンスの式を用いて、温度と相対湿度から絶対湿度（$g/m^3$）を算出。
- **視認性の高い表示 (240x240 ST7789 TFT)**: 
  - 時刻（白）、温度（オレンジ）、相対湿度（青）、絶対湿度（緑）を色分けして表示。
- **WiFi接続**: 起動時に自動的に指定のアクセスポイントへ接続し、時刻同期を開始。

## ハードウェア構成

### 使用部品
- ESP32 Development Board
- AHT10 温湿度センサー
- ST7789 TFT ディスプレイ (240x240)

### ピンアサイン

#### AHT10 センサー (I2C)
| ESP32 Pin | AHT10 Pin | 機能 |
| :--- | :--- | :--- |
| GPIO 21 | SDA | データ |
| GPIO 22 | SCL | クロック |

#### ST7789 TFT ディスプレイ (SPI)
| ESP32 Pin | Display Pin | 機能 |
| :--- | :--- | :--- |
| GPIO 18 | SCL (SCLK) | SPI クロック |
| GPIO 23 | SDA (MOSI) | SPI データ |
| GPIO 2 | DC (RS) | データ/コマンド選択 |
| GPIO 4 | RES (RST) | リセット |
| GPIO 5 | BLK (LED) | バックライト |

## ソフトウェア環境

### 依存ライブラリ
- **LovyanGFX**: ディスプレイ制御およびフォント表示。
- **Adafruit AHTX0**: AHT10 センサーのデータ取得。
- **WiFi / time**: WiFi 接続および NTP 時刻同期。

### 設定項目
`src/main.cpp` 内の以下の変数を環境に合わせて変更してください：
- `ssid`: Wi-Fi の SSID
- `password`: Wi-Fi のパスワード

## 使用方法
1. PlatformIO でプロジェクトを開きます。
2. 回路を接続し、ビルド・書き込みを行います。
3. 起動時、画面に `WiFi connecting...` と表示され、接続完了後にデジタル時計と環境データが表示されます。
